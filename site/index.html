Try it now

Install the packaged Fedora beta.

Download setup.sh and the Fedora RPM from the same GitHub release, then run the stable setup entrypoint. 

bash ./setup.sh --package ./operance-0.1.0-1.noarch.rpm
operance --installed-smoke
operance --supported-commands --supported-commands-available-only

Say what you want done.

open browser open google.com open firefox open firefox and notify me what time is it wifi status what is the volume

Boundary first

One core. Many operating systems.

Operance keeps intent, planner, policy, validation, daemon, and MCP logic portable. Platform providers decide host readiness. Adapters execute native desktop actions.
